## Usage

### Prepare a claim

```
npm run claim:prepare
```

### Review a claim

```
npm run claim:review
```

### Endorse a claim

```
npm run claim:endorse
```

### Test the Registry

```
npm run registry:test
```

### Update the Registry

```
npm run registry:update
```

### Drop the Registry

```
npm run registry:drop
```
